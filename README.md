# TaskControl
## Минималистичное Web-приложение, реализующее простой интерфейс для менеджмента задач.
В приложении реализованы прецеденты для взаимодействия с задачами: Назначение задач и подзадач, удаление, редактирование, запуск, приостановка, завершение.
